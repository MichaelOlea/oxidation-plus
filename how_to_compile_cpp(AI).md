# C++ Quick Start Guide (g++)

### 1. Create

Write your code and save it as a file ending in **.cpp** (e.g., `main.cpp`).

### 2. Compile

Open your terminal and run:
**g++ main.cpp -o my_app**
**g++ -std=c++23 main.cpp -o my_app** (speficy which version to use)

### 3. Run

* **Windows:** `my_app.exe`
* **Mac/Linux:** `./my_app`

---

**Fast Track (Compile & Run in one line):**

* **Windows:** `g++ main.cpp -o my_app && my_app`
* **Mac/Linux:** `g++ main.cpp -o my_app && ./my_app`
